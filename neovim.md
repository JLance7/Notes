| file tree |
| `nvim` |  neovim  |
| `ctrl+h` | open file explorer  |
| `space+t+h` | change theme  |
| `m` | mark file  |
| `a` | create new file |
| `c` | copy file |
| `p` | paste file |
| `r` | rename file |
| editor |
| `space+f+f` | find files |
| `space+f+b` | search open files |
| `space+c+h` | cheat sheet |
| `space` | hints |
| `i` | insert mode |
| `esc` | command mode |
| navigation |
| `ctrl+h` | go to file tree |
| `ctrl+L` | go to editor |
| `:vsp` | vertical split  |
| `:sp` | split |
| `space+n` | line numbers |
| `space+r+n` | relative line numbers |
| `tab` | switch tabs|
| `space+x` | close tab  |
| `:cd path` | open window in new folder  |
| terminal |
| `space+h` | open terminal |
| `space+v` | vertical terminal |
| `ctrl+x then ctrl+k` | go back to editor |
| `esc then ctrl+j` | go to terminal from editor |
